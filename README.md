## What is it?
EmicoBot is a Twitch chat bot designed for [Emico Mirari's Twitch stream](https://www.twitch.tv/emicomirari). Other chat bots exist, but oftentimes a lot of their features are locked behind a premium paywall. EmicoBot exists to gain full control over the bot and its commands, allowing for deeper chat integration and engagement.
